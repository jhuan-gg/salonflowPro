# Salonflow Pro

Aplicação de gestão para salões: agendamento, clientes, atendentes e serviços.

## Para que serve

- Gestão de agendamentos e histórico de atendimentos.
- Cadastro e gerenciamento de clientes, atendentes e serviços.

## Tecnologias principais

- Vite + React + TypeScript
- Tailwind CSS
- Supabase (integração para backend/BD)

## Como rodar localmente

1. Instale dependências:

```bash
npm install
```

2. Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

3. Abra no navegador em `http://localhost:5173` (padrão Vite).

Outros comandos úteis:

```bash
# Build de produção
npm run build

# Servir build localmente
npm run preview
```

Se o projeto usar variáveis de ambiente (ex: Supabase), crie um arquivo `.env` com as chaves necessárias antes de rodar.
